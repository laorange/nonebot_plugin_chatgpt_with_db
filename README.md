<div align="center"><h1>ChatGPT私聊机器人</h1></div>

## 特点

+ 可在好友私聊中使用，而不是在群里，减少打扰
+ 可在临时会话中使用，避免频繁添加好友的风险
+ 使用数据库记录各用户的使用量，便于消费统计

## 依赖库

|     名称     |     作用      |
| :----------: | :-----------: |
|   nonebot2   |  机器人框架   |
|   pydantic   |   类型检查    |
|    openai    | 与ChatGPT交互 |
| tortoise-orm |  建立数据库   |
